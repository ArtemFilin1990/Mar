# Краткий справочник

## Быстрый выбор подшипника

### По типу нагрузки

| Тип нагрузки | Рекомендуемый тип подшипника |
|--------------|------------------------------|
| Только радиальная | Шариковый радиальный однорядный |
| Радиальная + малая осевая | Шариковый радиальный однорядный |
| Радиальная + осевая средняя | Радиально-упорный шариковый |
| Радиальная + осевая большая | Конический роликовый |
| Только осевая | Упорный шариковый |
| Осевая большая | Упорный роликовый |
| Знакопеременная нагрузка | Цилиндрический роликовый |

### По скорости вращения

| Скорость (об/мин) | Рекомендуемый тип |
|-------------------|-------------------|
| До 3000 | Любой тип |
| 3000-10000 | Шариковые подшипники |
| Свыше 10000 | Радиальные шариковые с сепаратором |

### По условиям эксплуатации

| Условия | Рекомендация |
|---------|--------------|
| Высокие температуры (>150°C) | Высокотемпературные подшипники, специальная смазка |
| Загрязненная среда | Подшипники с уплотнениями (2RS, 2Z) |
| Влажная среда | Нержавеющие подшипники |
| Точное позиционирование | Класс точности 5 или выше |
| Низкий шум | Подшипники с низким уровнем вибрации |

## Монтаж подшипников

### Посадки на вал

| Условия работы | Посадка |
|----------------|---------|
| Внутреннее кольцо вращается | Плотная (k6, m6, n6) |
| Внутреннее кольцо неподвижно | Свободная (g6, h6) |
| Ударные нагрузки | Особо плотная (p6, r6) |

### Посадки в корпус

| Условия работы | Посадка |
|----------------|---------|
| Наружное кольцо вращается | Плотная (M7, N7) |
| Наружное кольцо неподвижно | Свободная (H7, G7) |
| Регулировка радиального зазора | Плотная (N7, P7) |

### Методы монтажа

1. **Холодная посадка**
   - Для свободных посадок
   - Использование пресса
   - Равномерное приложение усилия

2. **Горячая посадка**
   - Для плотных посадок
   - Нагрев подшипника до 80-100°C
   - Быстрая установка на вал

3. **С помощью втулок**
   - Закрепительные втулки (H-серия)
   - Стяжные втулки (AH-серия)
   - Не требуют нагрева

## Смазка подшипников

### Выбор типа смазки

| Условия | Тип смазки |
|---------|------------|
| Низкие скорости | Пластичная смазка |
| Высокие скорости (>3000 об/мин) | Масло |
| Высокие температуры | Синтетическая смазка |
| Низкие температуры | Низкотемпературная смазка |
| Загрязненная среда | Пластичная с уплотнениями |
| Пищевое производство | Пищевая смазка |

### Количество смазки

**Для пластичной смазки:**
- Заполнение 30-50% свободного объема подшипника
- При низких скоростях - до 70%
- При высоких скоростях - 30-40%

**Периодичность смазки:**
- Определяется по формуле или таблицам производителя
- Зависит от скорости, температуры, нагрузки

## Диагностика неисправностей

### Подшипники

| Симптом | Возможная причина | Решение |
|---------|-------------------|---------|
| Повышенный шум | Износ, недостаток смазки | Проверить смазку, заменить |
| Перегрев | Плотная посадка, избыток смазки | Проверить посадку, удалить лишнюю смазку |
| Вибрация | Износ, дефект монтажа | Проверить центровку, заменить |
| Заедание | Загрязнение, коррозия | Очистить, заменить |

### Ремни

| Симптом | Возможная причина | Решение |
|---------|-------------------|---------|
| Проскальзывание | Слабое натяжение | Увеличить натяжение |
| Быстрый износ | Перенатяжение, перекос | Отрегулировать натяжение и соосность |
| Трещины | Старение, перегрев | Заменить ремень |
| Шум | Износ, резонанс | Проверить состояние, заменить |

### Уплотнения

| Симптом | Возможная причина | Решение |
|---------|-------------------|---------|
| Утечка | Износ, повреждение | Заменить уплотнение |
| Затвердевание | Высокая температура | Использовать термостойкий материал |
| Набухание | Несовместимость со средой | Выбрать подходящий материал |

## Хранение

### Подшипники

- Хранить в заводской упаковке
- Температура: 5-25°C
- Влажность: <60%
- Избегать вибраций
- Вертикальное положение для крупных подшипников

### Ремни

- Хранить вдали от источников тепла и озона
- Избегать прямых солнечных лучей
- Подвешивать на стержне или хранить горизонтально
- Не допускать изгибов и перекручивания

### Рукава

- Хранить в сухом помещении
- Избегать УФ-излучения
- Защищать заглушками
- Не хранить под грузом
- Срок хранения: обычно до 5 лет

### Уплотнения

- Хранить в закрытой упаковке
- Защищать от озона и УФ-излучения
- Температура: 15-25°C
- Срок хранения: обычно до 5 лет
- Не допускать деформации

## Безопасность

### При монтаже

⚠️ **Внимание:**
- Использовать средства индивидуальной защиты
- Не превышать допустимую температуру нагрева
- Использовать правильный инструмент
- Не ударять по подшипнику

### При эксплуатации

⚠️ **Внимание:**
- Регулярно проверять состояние
- Не превышать допустимые нагрузки
- Следить за температурой
- Защищать от загрязнений

### При работе с гидравликой

⚠️ **Внимание:**
- Сбросить давление перед разборкой
- Использовать защитные очки
- Проверять состояние рукавов
- Соблюдать чистоту

# 4. Аналоги ГОСТ ↔ ISO

## 4.1 Принципы подбора аналогов

При подборе аналогов между системами ГОСТ и ISO необходимо учитывать:

1. **Основные размеры** (d, D, B) должны совпадать
2. **Тип подшипника** должен быть идентичным
3. **Класс точности** может отличаться обозначением, но не характеристиками
4. **Радиальный зазор** обозначается по-разному, но значения совпадают

> **Важно:** Всегда сверяйте размеры по чертежам или справочникам перед заменой!

## 4.7 Примеры подбора аналогов

### Пример 1: ГОСТ → ISO

**Исходный:** 180205-Ш-С3  
**Требуется:** найти аналог ISO

**Решение:**
1. 180205 → базовое обозначение серии 180200
2. По таблице: 180205 = 6205
3. Ш → RS (резиновое уплотнение)
4. С3 → C3 (увеличенный зазор)

**Результат:** 6205-RS-C3

### Пример 2: ISO → ГОСТ

**Исходный:** 7208-B-P5  
**Требуется:** найти аналог ГОСТ

**Решение:**
1. 7208 → радиально-упорный шариковый
2. Серия 7200 → ГОСТ серия 66200
3. 7208 = 66208
4. B → угол 40° (соответствует стандарту)
5. P5 → класс 5

**Результат:** 66208-5

### Пример 3: С уплотнениями

**Исходный:** 6206-2RS-C3  
**Требуется:** найти аналог ГОСТ

**Решение:**
1. 6206 → серия 6200 → ГОСТ 180200
2. По таблице: 6206 = 180206
3. 2RS → 2Ш (два уплотнения)
4. C3 → С3

**Результат:** 180206-2Ш-С3

## 4.8 Практические рекомендации

### При замене ГОСТ на ISO:
✓ Проверьте совпадение основных размеров  
✓ Убедитесь в соответствии класса точности  
✓ Учитывайте требования к радиальному зазору  
✓ Сверьте тип уплотнений при их наличии

### При замене ISO на ГОСТ:
✓ Найдите подшипник с идентичными размерами  
✓ Проверьте доступность в России  
✓ Учитывайте возможную разницу в цене  
✓ Убедитесь в совместимости посадок

> **Примечание:** В случае сомнений всегда обращайтесь к справочникам производителей или техническим специалистам.

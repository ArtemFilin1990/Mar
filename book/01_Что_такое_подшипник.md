# 1. Что такое подшипник

## 1.1 Определение и назначение

**Подшипник** — это опора или направляющая вала, оси или иного подвижного элемента машины, которая фиксирует его положение в пространстве и воспринимает нагрузки при вращении или возвратно-поступательном движении.

### Основные функции подшипника:
- Поддержка вращающихся или качающихся деталей
- Уменьшение трения между движущимися частями
- Обеспечение точного позиционирования вала
- Восприятие радиальных и осевых нагрузок

## 1.2 Типы подшипников

### По принципу работы

Подшипники делятся на два основных класса:

#### Подшипники качения
Между кольцами находятся тела качения (шарики или ролики), которые катятся по дорожкам качения.

**Преимущества:**
- Малое трение
- Низкий нагрев
- Не требуют сложной системы смазки
- Взаимозаменяемость (стандартизация)

**Недостатки:**
- Чувствительность к ударным нагрузкам
- Шум при работе на высоких скоростях
- Ограниченный ресурс

#### Подшипники скольжения
Вал вращается в отверстии втулки или корпуса с малым зазором, между ними — смазка.

**Преимущества:**
- Бесшумность работы
- Способность выдерживать ударные нагрузки
- Компактность конструкции

**Недостатки:**
- Требуют постоянной подачи смазки
- Больший момент трения (особенно при пуске)
- Нужна точная подгонка по размерам

## 1.3 Устройство подшипника качения

Типичный подшипник качения состоит из следующих элементов:

```
    Наружное кольцо
         ↓
    ┌────────┐
    │ ○  ○  ○│ ← Тела качения (шарики/ролики)
    │  ╱│╲  │
    └──┴─┴──┘
       ↑
  Внутреннее кольцо
```

### Основные элементы:

| Элемент | Назначение |
|---------|------------|
| **Внутреннее кольцо** | Устанавливается на вал, имеет дорожку качения |
| **Наружное кольцо** | Устанавливается в корпус, имеет дорожку качения |
| **Тела качения** | Шарики или ролики, воспринимают нагрузку |
| **Сепаратор (обойма)** | Удерживает тела качения на равном расстоянии |
| **Уплотнения/защиты** | Предотвращают попадание загрязнений (опционально) |

## 1.4 Классификация подшипников качения

### По типу тел качения

- **Шариковые** — используют шарики
  - Радиальные
  - Радиально-упорные
  - Упорные
  
- **Роликовые** — используют ролики
  - С цилиндрическими роликами
  - С коническими роликами
  - Со сферическими роликами
  - Игольчатые

### По направлению воспринимаемой нагрузки

| Тип | Нагрузка | Применение |
|-----|----------|------------|
| **Радиальные** | Перпендикулярно оси вала | Общее машиностроение |
| **Упорные** | Вдоль оси вала | Вертикальные валы, винты |
| **Радиально-упорные** | Комбинированная | Коробки передач, редукторы |

### По количеству рядов тел качения

- Однорядные
- Двухрядные
- Многорядные

## 1.5 Основные параметры подшипников

### Геометрические размеры

- **d** — внутренний диаметр (диаметр отверстия)
- **D** — наружный диаметр
- **B** (или H, T) — ширина подшипника

> **Важно:** Подшипники стандартизованы по размерам согласно ГОСТ и ISO.

### Грузоподъемность

**Динамическая грузоподъемность (C)** — нагрузка, которую подшипник может выдержать при вращении в течение 1 миллиона оборотов с вероятностью отказа не более 10%.

**Статическая грузоподъемность (C₀)** — нагрузка, при которой в наиболее нагруженной точке контакта остаточная деформация не превысит 0,0001 диаметра тела качения.

### Класс точности

Определяет допуски на размеры и форму элементов подшипника:

| Класс ГОСТ | Класс ISO | Применение |
|------------|-----------|------------|
| 0 (нормальный) | P0 | Общее машиностроение |
| 6 | P6 | Повышенные требования |
| 5 | P5 | Станки, точные механизмы |
| 4 | P4 | Высокоточные станки |
| 2 | P2 | Особо точные приборы |

### Радиальный зазор

Расстояние, на которое одно кольцо может перемещаться относительно другого в радиальном направлении.

Обозначения групп зазоров: **C2**, **C0** (нормальный), **C3**, **C4**, **C5**

## 1.6 Маркировка подшипников

Подшипники маркируются кодом, который содержит информацию о типе, размерах и дополнительных характеристиках.

### Пример расшифровки (ISO):

```
6205-2RS-C3
│││││ │  └─ C3 — увеличенный радиальный зазор
│││││ └─ 2RS — резиновые уплотнения с двух сторон
││││└─ 05 — диаметр отверстия 25 мм (05 × 5)
│││└─ 2 — серия диаметров (средняя)
││└─ 0 — тип подшипника (радиальный шариковый)
│└─ 6 — тип подшипника по ISO
└─ базовое обозначение
```

Подробнее о системах обозначений см. главы:

- **[ГОСТ Подшипники](03_ГОСТ_Подшипники.md)**

## 1.7 Материалы подшипников

### Стандартные подшипники

- **Кольца и тела качения:** сталь ШХ15 (аналог 100Cr6, AISI 52100)
  - Твердость: 60-65 HRC после закалки
  - Высокая износостойкость

- **Сепараторы:**
  - Стальные штампованные
  - Латунные массивные (для высоких нагрузок)
  - Полиамидные (для низких температур и химстойкости)

### Специальные подшипники

- **Керамические (гибридные):** тела качения из Si₃N₄ (нитрид кремния)
  - Меньший вес
  - Выше скорость вращения
  - Не подвержены коррозии

- **Нержавеющая сталь:** для агрессивных сред и пищевой промышленности

## 1.8 Смазка подшипников

### Типы смазки

**Пластичная смазка (консистентная):**
- Литол-24, Солидол, ЦИАТИМ-201
- Удобна в применении
- Не требует сложной системы подачи

**Жидкая смазка (масло):**
- Для высоких скоростей
- Лучше отводит тепло
- Требует системы циркуляции

### Уплотнения

- **Z** — металлический щиток (защита с одной стороны)
- **2Z** — щитки с двух сторон
- **RS** — резиновое уплотнение (контактное)
- **2RS** — уплотнения с двух сторон

> **Примечание:** Подшипники с уплотнениями обычно поставляются со смазкой на весь срок службы.

# 99. Приложения

## Глоссарий

### А

**Асбест** - волокнистый минерал, используемый в уплотнительных материалах (шнуры, набивки). Обладает высокой термостойкостью.

**Ацетилен** - горючий газ для газовой сварки и резки металлов. Подается через специальные красные рукава.

### Б

**Борт** - выступающая часть на кольце подшипника для восприятия осевых нагрузок.

**БРС (Быстроразъемное соединение)** - устройство для быстрого подключения/отключения гидравлических или пневматических линий без инструмента.

**Бронза** - сплав меди с оловом или другими элементами, используемый для изготовления втулок скольжения.

### В

**Вал** - вращающаяся деталь, передающая крутящий момент. На вал устанавливается внутреннее кольцо подшипника.

**Вариатор** - бесступенчатая передача с плавным изменением передаточного отношения.

**Витон (FKM)** - фторкаучук, материал уплотнений для высоких температур и агрессивных сред.

**Втулка** - цилиндрическая деталь для установки в отверстие. Может быть подшипником скольжения или элементом крепления.

**Втулка закрепительная (H-серия)** - коническая разрезная втулка для крепления подшипников с коническим отверстием на цилиндрических валах.

### Г

**Гибридный подшипник** - подшипник с керамическими телами качения и стальными кольцами.

**ГОСТ** - Государственный стандарт, система технических стандартов в России и СНГ.

**Грузоподъемность динамическая** - нагрузка, которую подшипник может выдержать при определенной долговечности (обычно 1 млн оборотов).

**Грузоподъемность статическая** - максимальная нагрузка в неподвижном состоянии без остаточной деформации.

### З

**Зазор радиальный** - расстояние между телами качения и дорожкой качения в ненагруженном состоянии.

**Закалка** - термообработка стали для повышения твердости и износостойкости.

### И

**Игольчатый подшипник** - роликовый подшипник с тонкими длинными роликами (иглами).

**ISO** - Международная организация по стандартизации.

### К

**Класс точности** - категория подшипника по допускам на размеры и форму элементов.

**Клиновой ремень** - приводной ремень с трапецеидальным сечением.

**Конический роликовый подшипник** - подшипник с коническими роликами, воспринимает радиальные и осевые нагрузки.

### М

**Манжета** - уплотнительный элемент для подвижных соединений (валов, штоков).

### Н

**NBR** - бутадиен-нитрильный каучук, наиболее распространенный материал уплотнений для масел.

### О

**О-ринг (O-ring)** - уплотнительное кольцо круглого сечения.

**Осевая нагрузка** - нагрузка, действующая вдоль оси вращения.

### П

**Паронит** - листовой прокладочный материал на основе асбеста с каучуковым связующим.

**Подшипник качения** - подшипник, работающий на основе качения тел (шариков или роликов).

**Подшипник скольжения** - подшипник, работающий на основе скольжения вала по втулке.

**Поликлиновой ремень** - ремень с несколькими продольными клиньями на внутренней поверхности.

**PTFE** - политетрафторэтилен (тефлон), материал с очень низким коэффициентом трения.

### Р

**Радиальная нагрузка** - нагрузка, действующая перпендикулярно оси вращения.

**Ремень** - гибкий элемент для передачи вращения между шкивами.

**РТИ** - резинотехнические изделия.

### С

**Сепаратор** - деталь подшипника, удерживающая тела качения на равном расстоянии.

**Синхронный ремень** - зубчатый ремень для передачи вращения без проскальзывания.

### Т

**Тело качения** - шарик или ролик, передающий нагрузку в подшипнике качения.

### У

**Уплотнение** - элемент для предотвращения утечки жидкости или газа.

**Упорный подшипник** - подшипник для восприятия осевых нагрузок.

### Ш

**Шарик** - сферическое тело качения в шариковом подшипнике.

**Шкив** - колесо с канавками для ремня.

## Сокращения и аббревиатуры

- **БРС** - Быстроразъемное соединение
- **ГОСТ** - Государственный стандарт
- **ISO** - International Organization for Standardization
- **РВД** - Рукав высокого давления
- **РТИ** - Резинотехнические изделия
- **ЧПУ** - Числовое программное управление

### Материалы уплотнений

- **NBR** - Nitrile Butadiene Rubber (бутадиен-нитрильный каучук)
- **EPDM** - Ethylene Propylene Diene Monomer (этилен-пропиленовый каучук)
- **FKM** - Fluorocarbon (фторкаучук, витон)
- **PTFE** - Polytetrafluoroethylene (политетрафторэтилен, тефлон)

### Суффиксы подшипников

- **RS** - Rubber Seal (резиновое уплотнение)
- **2RS** - два резиновых уплотнения
- **Z** - Metal shield (металлическая защита)
- **2Z** - две металлические защиты
- **K** - Tapered bore (коническое отверстие)
- **N** - Snap ring groove (канавка для стопорного кольца)

## Полезные формулы

### Расчет грузоподъемности

**Эквивалентная нагрузка:**

```
P = X·Fr + Y·Fa
```

Где:
- **P** - эквивалентная нагрузка
- **Fr** - радиальная нагрузка
- **Fa** - осевая нагрузка
- **X, Y** - коэффициенты (из справочников)

### Расчет ресурса подшипника

**Базовый расчетный ресурс (L₁₀):**

```
L₁₀ = (C/P)ᵖ
```

Где:
- **L₁₀** - ресурс в млн оборотов
- **C** - динамическая грузоподъемность
- **P** - эквивалентная нагрузка
- **p** - показатель степени (3 для шариковых, 10/3 для роликовых)

### Ресурс в часах

```
L₁₀h = (L₁₀ × 10⁶) / (60 × n)
```

Где:
- **n** - частота вращения (об/мин)

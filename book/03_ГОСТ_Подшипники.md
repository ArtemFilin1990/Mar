# 3. Подшипники по ГОСТ

## 3.1 Система обозначений по ГОСТ

Отечественная система обозначений подшипников регламентируется **ГОСТ 3189-89** "Подшипники шариковые и роликовые. Система условных обозначений".

## 3.2 Структура обозначения

Базовое обозначение подшипника ГОСТ:

```
1 8 0 2 05
│ │ │ │ └─ Внутренний диаметр
│ │ │ └─ Серия диаметров
│ │ └─ Тип подшипника
│ └─ Серия ширин
└─ Тип подшипника (первая цифра)
```

### Полное обозначение с суффиксами:

```
180205-Ш-С3
│││││││  │ └─ Суффикс (радиальный зазор)
│││││││  └─ Суффикс (уплотнения)
│││││││
│││││││  Базовое обозначение:
││││││└─ 05 — код внутреннего диаметра
│││││└─ 2 — серия диаметров
││││└─ 0 — тип подшипника
│││└─ 8 — серия ширин
││└─ 0 — дополнительный код
│└─ 1 — тип подшипника
└─ Полное обозначение
```

## 3.3 Типы подшипников (первые цифры)

| Код | Тип подшипника | Пример |
|-----|----------------|--------|
| **0** | Радиальный шариковый однорядный | 205 |
| **1** | Радиальный шариковый сферический двухрядный | 1205 |
| **2** | Роликовый с короткими цилиндрическими роликами | 2205 |
| **3** | Роликовый сферический двухрядный | 3205 |
| **4** | Игольчатый | 4205 |
| **5** | Роликовый с витыми роликами | 5205 |
| **6** | Радиально-упорный шариковый | 6-205 |
| **7** | Роликовый конический | 7205 |
| **8** | Упорный шариковый | 8105 |
| **9** | Упорный роликовый | 9115 |

## 3.4 Серия ширин (вторая цифра для легкой и средней серий)

| Код | Серия | Характеристика |
|-----|-------|----------------|
| **1** | Особо легкая | Сверхлегкие конструкции |
| **2** | Легкая | Стандартные применения |
| **3** | Средняя | Повышенные нагрузки |
| **4** | Тяжелая | Высокие нагрузки |

## 3.5 Серии диаметров (третья цифра)

| Серия | Размер наружного диаметра |
|-------|---------------------------|
| **0, 1** | Особо легкая |
| **2** | Легкая |
| **3** | Средняя |
| **4** | Тяжелая |
| **5, 6** | Легкая широкая |

## 3.6 Код внутреннего диаметра

### Для d от 20 до 495 мм:

**Диаметр = код × 5 мм**

| Код | d (мм) | Код | d (мм) | Код | d (мм) |
|-----|--------|-----|--------|-----|--------|
| 00 | 10 | 05 | 25 | 10 | 50 |
| 01 | 12 | 06 | 30 | 12 | 60 |
| 02 | 15 | 07 | 35 | 14 | 70 |
| 03 | 17 | 08 | 40 | 16 | 80 |
| 04 | 20 | 09 | 45 | 18 | 90 |

### Для d < 10 мм:

Диаметр указывается непосредственно через дробь (например, 60/8 — d = 8 мм).

### Для d > 500 мм:

После слэша указывается полный размер (например, 60/530 — d = 530 мм).

## 3.7 Классы точности по ГОСТ

| Класс ГОСТ | Класс ISO | Применение |
|------------|-----------|------------|
| **0** | P0 | Нормальная точность, общее машиностроение |
| **6** | P6 | Повышенная точность |
| **5** | P5 | Высокая точность, станки |
| **4** | P4 | Особо высокая точность |
| **2** | P2 | Прецизионные механизмы |
| **Т** | — | Точные подшипники для авиации |

## 3.8 Группы радиального зазора

| Обозначение ГОСТ | Характеристика |
|------------------|----------------|
| **6** | Меньше С2 |
| **7** | Между С2 и нормальным |
| **C (норм.)** | Нормальный зазор |
| **8** | Между нормальным и С3 |
| **С3** | Больше нормального |
| **С4** | Увеличенный |
| **С5** | Значительно увеличенный |

## 3.9 Дополнительные знаки

### Материал сепаратора

| Суффикс | Материал |
|---------|----------|
| **Р** | Сепаратор из полиамида |
| **Г** | Сепаратор из фенольных смол |
| **Е** | Сепаратор из эбонита |
| **Л** | Сепаратор из латуни |
| **Д** | Сепаратор из дюралюминия |

### Уплотнения и защиты

| Суффикс | Описание |
|---------|----------|
| **З** | Защитная шайба с одной стороны |
| **2З** | Защитные шайбы с двух сторон |
| **Ш** | Уплотнение контактное резиновое с одной стороны |
| **2Ш** | Уплотнения контактные резиновые с двух сторон |
| **Н** | Уплотнение бесконтактное с одной стороны |
| **2Н** | Уплотнения бесконтактные с двух сторон |

### Момент трения

| Суффикс | Категория |
|---------|-----------|
| **М** | Малый момент трения |
| **МУ** | Малый момент трения, улучшенное качество |

### Конструктивные особенности

| Суффикс | Описание |
|---------|----------|
| **К** | Коническое отверстие (1:12) |
| **Е** | Подшипник без сепаратора |
| **У** | Подшипник упрочненный |
| **В** | Высокое качество |

## 3.10 Примеры расшифровки

### Пример 1: **180205-Ш-С3**

```
180205-Ш-С3
│││││││  │ └─ С3 — увеличенный радиальный зазор
│││││││  └─ Ш — резиновое уплотнение с одной стороны
││││││└─ 05 — диаметр 25 мм (5 × 5)
│││││└─ 2 — серия диаметров (легкая)
││││└─ 0 — радиальный шариковый
│││└─ 8 — серия ширин (легкая)
││└─ 0 — дополнительный код
│└─ 1 — тип подшипника
└─ базовое обозначение
```

**Характеристики:**
- Радиальный шариковый однорядный
- d = 25 мм, D = 52 мм, B = 15 мм
- Уплотнение с одной стороны
- Увеличенный радиальный зазор
- **Аналог ISO:** 6205-RS-C3

### Пример 2: **7208-А**

```
7208-А
│││││ └─ А — угол контакта 12°
││││└─ 08 — диаметр 40 мм
│││└─ 2 — серия диаметров
││└─ 0 — конический роликовый
│└─ 7 — тип подшипника
└─ базовое обозначение
```

**Характеристики:**
- Роликовый конический
- d = 40 мм
- Угол контакта 12°
- **Аналог ISO:** 30208

### Пример 3: **32306-А-Р**

```
32306-А-Р
│││││││ │ └─ Р — сепаратор полиамидный
│││││││ └─ А — угол контакта
││││││└─ 06 — диаметр 30 мм
│││││└─ 3 — серия диаметров (средняя)
││││└─ 2 — роликовый сферический
│││└─ 3 — серия ширин
││└─ дополнительный код
│└─ тип подшипника
└─ базовое обозначение
```

**Характеристики:**
- Роликовый сферический двухрядный
- d = 30 мм
- Сепаратор из полиамида
- **Аналог ISO:** 22206-E

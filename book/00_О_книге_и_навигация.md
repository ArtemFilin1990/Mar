# О книге и навигация

## Описание

Данная книга представляет собой систематизированное изложение материалов по подшипникам и элементам механических передач. Материал основан на технических каталогах и стандартах ГОСТ/ISO.

## Структура книги

Книга разделена на логические главы, каждая из которых охватывает определенную тематическую область:

### Основные разделы

1. **[Что такое подшипник](01_Что_такое_подшипник.md)** - Базовые понятия, классификация, устройство
2. **[Подшипники по ISO](02_ISO_Подшипники.md)** - Международная система обозначений
3. **[Подшипники по ГОСТ](03_ГОСТ_Подшипники.md)** - Отечественная система обозначений
4. **[Аналоги ГОСТ ↔ ISO](04_Аналоги_ГОСТ_ISO.md)** - Таблицы соответствия и подбора аналогов
5. **[Ремни приводные](05_Ремни.md)** - Клиновые, поликлиновые, синхронные ремни
6. **[РТИ и уплотнения](06_РТИ_и_уплотнения.md)** - Резинотехнические изделия, манжеты, О-ринги
7. **[Приложения](99_Приложения.md)** - Глоссарий, справочные таблицы, стандарты

## Как пользоваться книгой

### Для изучения с нуля
Рекомендуется читать последовательно, начиная с главы 1.

### Для быстрого поиска
Используйте:
- **Оглавление** в каждой главе для навигации к нужному подразделу
- **Навигационные ссылки** внизу каждой страницы для перехода между главами
- **[Глоссарий](99_Приложения.md#глоссарий)** для поиска терминов

### Для подбора аналогов
Перейдите в главу **[Аналоги ГОСТ ↔ ISO](04_Аналоги_ГОСТ_ISO.md)** и используйте таблицы соответствия.

## Соглашения по оформлению

- **Жирный текст** — важные термины и ключевые понятия
- *Курсив* — примеры, пояснения
- `Моноширинный` — коды обозначений подшипников
- Таблицы — для технических данных и соответствий

> **Примечание:** Все размеры указаны в миллиметрах, если не указано иное.

## Источники

Материалы книги составлены на основе:
- Технических каталогов производителей подшипников (SKF, FAG, NSK, NTN, TIMKEN и др.)
- Стандартов ГОСТ (ГОСТы серии 520, 3189, 8338 и др.)
- Стандартов ISO (ISO 15, ISO 492 и др.)
- Справочной литературы по подшипникам и механическим передачам

# 2. Подшипники по ISO

## 2.1 Международная система обозначений ISO

Система обозначений ISO (International Organization for Standardization) является наиболее распространенной в мире и используется большинством производителей подшипников.

## 2.2 Структура обозначения

Базовое обозначение подшипника ISO состоит из:

```
6 2 05
│ │ └─ Код внутреннего диаметра
│ └─ Серия диаметров
└─ Тип подшипника
```

### Полное обозначение с суффиксами:

```
6205-2RS1-C3
│││││  │  └─ Суффикс (радиальный зазор)
│││││  └─ Суффикс (уплотнения)
│││││
│││││  Базовое обозначение:
││││└─ 05 — код внутреннего диаметра
│││└─ 2 — серия диаметров
││└─ 0 — тип (радиальный шариковый однорядный)
│└─ 6 — класс подшипника
└─ Полное обозначение
```

## 2.3 Типы подшипников (первая цифра)

| Код | Тип подшипника | Пример |
|-----|----------------|--------|
| **6** | Радиальный шариковый однорядный | 6205 |
| **7** | Радиально-упорный шариковый | 7210 |
| **3** | Радиально-упорный шариковый двухрядный | 3205 |
| **4** | Радиальный шариковый двухрядный | 4205 |
| **2** | Сферический роликовый с бочкообразными роликами | 22205 |
| **N** | Цилиндрический роликовый | NU205 |
| **3** | Конический роликовый | 30205 |
| **5** | Упорный шариковый | 51105 |
| **2** | Упорный роликовый сферический | 29315 |

## 2.4 Серии диаметров (вторая цифра)

Определяет соотношение между наружным и внутренним диаметрами:

| Серия | Характеристика | Применение |
|-------|----------------|------------|
| **0, 1** | Экстра-легкая | Приборы, малые нагрузки |
| **2** | Легкая | Стандартное применение |
| **3** | Средняя | Повышенные нагрузки |
| **4** | Тяжелая | Высокие нагрузки |

### Примеры серий для радиальных шариковых подшипников:

| Обозначение | Серия | d (мм) | D (мм) | B (мм) |
|-------------|-------|--------|--------|--------|
| 6005 | 00 | 25 | 47 | 12 |
| 6205 | 02 | 25 | 52 | 15 |
| 6305 | 03 | 25 | 62 | 17 |
| 6405 | 04 | 25 | 80 | 21 |

## 2.5 Код внутреннего диаметра

### Для подшипников с d от 20 мм:

**Диаметр отверстия = код × 5 мм**

| Код | d (мм) | Код | d (мм) | Код | d (мм) |
|-----|--------|-----|--------|-----|--------|
| 00 | 10 | 05 | 25 | 10 | 50 |
| 01 | 12 | 06 | 30 | 12 | 60 |
| 02 | 15 | 07 | 35 | 14 | 70 |
| 03 | 17 | 08 | 40 | 16 | 80 |
| 04 | 20 | 09 | 45 | 18 | 90 |

### Для подшипников с d < 10 мм:

Внутренний диаметр указывается непосредственно (например, 618/8 — d = 8 мм).

### Для подшипников с d > 500 мм:

После слэша указывается полный размер (например, 60/500 — d = 500 мм).

## 2.6 Суффиксы и дополнительные обозначения

### Уплотнения и защиты

| Суффикс | Описание |
|---------|----------|
| **Z** | Металлический щиток с одной стороны |
| **2Z** | Металлические щитки с двух сторон |
| **RS** или **RS1** | Резиновое контактное уплотнение с одной стороны |
| **2RS** или **2RS1** | Резиновые контактные уплотнения с двух сторон |
| **RZ** | Резиновое бесконтактное уплотнение с одной стороны |
| **2RZ** | Резиновые бесконтактные уплотнения с двух сторон |

### Радиальный зазор

| Суффикс | Зазор |
|---------|-------|
| **C2** | Меньше нормального |
| **(нет)** | Нормальный (CN или C0) |
| **C3** | Больше нормального |
| **C4** | Увеличенный |
| **C5** | Значительно увеличенный |

### Класс точности

| Суффикс | Класс точности ISO | Аналог ГОСТ |
|---------|--------------------|-------------|
| **(нет)** | P0 | 0 (нормальный) |
| **P6** | P6 | 6 |
| **P5** | P5 | 5 |
| **P4** | P4 | 4 |
| **P2** | P2 | 2 |

### Материал сепаратора

| Суффикс | Материал |
|---------|----------|
| **M** | Латунный массивный сепаратор |
| **F** | Стальной штампованный сепаратор |
| **J** | Стальной штампованный сепаратор (другая конструкция) |
| **P** | Полиамидный сепаратор |
| **T** | Текстолитовый сепаратор |

### Другие обозначения

| Суффикс | Описание |
|---------|----------|
| **K** | Коническое отверстие (1:12) |
| **K30** | Коническое отверстие (1:30) |
| **N** | Канавка на наружном кольце для стопорного кольца |
| **NR** | Канавка со стопорным кольцом |
| **H** | Закрепительная втулка |
| **E** | Усиленная конструкция |

## 2.7 Примеры расшифровки

### Пример 1: **6205-2RS-C3**

```
6205-2RS-C3
││││  │  └─ C3 — радиальный зазор больше нормального
││││  └─ 2RS — резиновые уплотнения с двух сторон
│││└─ 05 — диаметр 25 мм (5 × 5)
││└─ 2 — серия диаметров (легкая)
│└─ 0 — радиальный шариковый однорядный
└─ 6 — тип по ISO
```

**Характеристики:**
- Тип: радиальный шариковый однорядный
- d = 25 мм, D = 52 мм, B = 15 мм
- Уплотнения с двух сторон
- Увеличенный радиальный зазор

### Пример 2: **7210-B-TVP-P5**

```
7210-B-TVP-P5
│││││  │  │  └─ P5 — класс точности P5
│││││  │  └─ TVP — специальная конструкция
│││││  └─ B — угол контакта 40°
││││└─ 10 — диаметр 50 мм
│││└─ 2 — серия диаметров
││└─ 1 — радиально-упорный
│└─ 7 — тип по ISO
└─ базовое обозначение
```

**Характеристики:**
- Тип: радиально-упорный шариковый
- d = 50 мм
- Угол контакта 40°
- Высокая точность (P5)

### Пример 3: **NU306-E-M1-C3**

```
NU306-E-M1-C3
│││││  │  │  └─ C3 — увеличенный зазор
│││││  │  └─ M1 — латунный сепаратор
│││││  └─ E — усиленная конструкция
││││└─ 06 — диаметр 30 мм
│││└─ 3 — серия диаметров (средняя)
││└─ 0 — цилиндрический роликовый
│└─ NU — тип без бортов на внутреннем кольце
└─ базовое обозначение
```

**Характеристики:**
- Тип: цилиндрический роликовый
- d = 30 мм
- Усиленная конструкция
- Латунный сепаратор
- Увеличенный зазор

Как ограничить доступ в Битрикс24 по IP-адресу
Автор: Наталья Штенникова
Последнее обновление: 16 июня 2025 г.
В Битрикс24 можно выбрать, с каких IP-адресов сотрудники смогут заходить на портал. Гибкая настройка поможет предотвратить попытки входа в ваш Битрикс24 с неподтвержденных IP-адресов.
Ограничивайте доступ по статическим и динамическим IP-адресам, если в них указан конкретный диапазон. Устанавливать ограничения доступа по IP-адресам может только сотрудник с правами администратора.
Ограничение доступа по IP-адресам работает не на всех тарифах.
Тарифные планы
Перейдите в
Настройки > Безопасность > Ограничение доступа по IP-адресам
Выберите, с каких IP-адресов сотрудники будут заходить в Битрикс24. Например, укажите для финансового отдела IP-адрес, закрепленный за офисом. Тогда сотрудники смогут работать в Битрикс24 только оттуда. Это поможет снизить риск утечки информации.
Чтобы добавить IP-адреса для других пользователей, нажмите кнопку внизу справа. Сохраните изменения.
Администратору вашего Битрикс24 нельзя ограничить доступ по IP-адресу.
Теперь сотрудники смогут зайти в Битрикс24 только с указанных IP-адресов.

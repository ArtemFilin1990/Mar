В списке CRM-форм собраны все формы, которые вы создали в Битрикс24. Здесь можно увидеть основную информацию, изменить настройки, оценить конверсию.

**Путь:** CRM > Продажи > Каналы продаж > CRM-формы

---

## Как настроить список CRM-форм

### Доступные колонки

| Колонка | Описание |
|---------|----------|
| **ID** | Уникальный номер формы (нельзя изменить) |
| **Название** | Внутреннее название для сотрудников |
| **Дата создания** | Когда создана форма |
| **Активировано** | Дата последнего изменения активности |
| **Кто активировал** | Сотрудник, изменивший активность |
| **Публичная ссылка** | Быстрый доступ к форме |
| **Код на сайте** | Варианты публикации формы |
| **Активность** | Включена/выключена форма |
| **Форма создает** | Какие элементы добавятся в CRM |
| **Конверсия** | Эффективность за 14 дней |
| **Изменения** | Динамика заполнений |
| **Форма с оплатой** | Есть ли возможность оплаты |

---

### Публичная ссылка

Нажмите **Открыть** для быстрого доступа:
- Посмотреть форму на мобильном и в браузере
- Заполнить форму
- Скопировать ссылку для отправки клиенту

---

### Код на сайте

Варианты публикации:
- На странице сайта
- В виджете
- В онлайн-чате
- Публичная ссылка

---

### Конверсия

Эффективность формы за последние 14 дней — процент клиентов, заполнивших форму после просмотра.

> Если показатель менее 50%, форму стоит доработать

---

## Как управлять CRM-формами

Нажмите **Меню (≡)** для действий:
- Редактировать
- Копировать (без настроек дизайна)
- Обнулить счетчики
- Удалить (без восстановления)

### Групповые действия

1. Отметьте нужные формы
2. Нажмите **Выбрать действие**
3. Выберите действие и нажмите **Применить**

---

## Коротко

- В списке собраны все CRM-формы Битрикс24
- Можно настроить отображаемые колонки
- Оценивайте конверсию и динамику заполнений
- Используйте групповые действия для массовых операций

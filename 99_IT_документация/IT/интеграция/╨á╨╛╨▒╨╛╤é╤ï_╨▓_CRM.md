Роботы — это инструмент для автоматизации рутинных задач. Они выполняют действия, когда элемент CRM попадает на определенную стадию.

> Роботы в CRM доступны не на всех тарифах

---

## Где работают роботы

- Лиды
- Сделки
- Предложения
- Счета
- Смарт-процессы

---

## Как добавить робота

1. Перейдите в раздел **CRM** и выберите элемент
2. Откройте вкладку **Роботы**
3. Нажмите **Создать**
4. Выберите стадию
5. Используйте **Фильтр > Роботы в Битрикс24**
6. Нажмите **Добавить** напротив нужного робота

> Чтобы добавить несколько роботов, зажмите Ctrl (Cmd на Mac)

## Настройки робота

### Общие настройки

| Настройка | Описание |
|-----------|----------|
| **По очереди** | После предыдущих или независимо |
| **Когда** | Сразу или через время |
| **При условии** | Только для определенных элементов |

### Индивидуальные настройки

Зависят от конкретного робота. Например:
- Товар, стоимость, количество
- Текст письма, получатель
- Поля для изменения

---

## Как проверить работу

Откройте карточку элемента CRM > вкладка **Роботы**

### Статусы

| Цвет | Иконка | Значение |
|------|--------|----------|
| Зеленый | ✓ | Робот сработал |
| Зеленый | ⚠ | Ошибка (наведите для подробностей) |
| Белый | ○ | Пауза, ждет выполнения |
| Белый | — | Не сработал, проверьте настройки |

---

## Управление роботами

| Действие | Описание |
|----------|----------|
| **Удалить** | Если робот больше не нужен |
| **Изменить** | Редактирование настроек |
| **Копировать** | Создание копии с настройками |
| **Выключить** | Временное отключение |
| **Остановить** | Прервать цепочку для элемента |

---

## Коротко

- Роботы автоматизируют рутинные задачи
- Работают при переходе на определенную стадию
- Группируются по типам задач
- Проверяйте статусы в карточке элемента
- Используйте отладчик для диагностики

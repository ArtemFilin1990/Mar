CRM-формы в Битрикс24 помогают собирать заявки, контакты и отзывы. Клиенты записываются на прием, бронируют услуги, оформляют заказы на сайте, а данные из формы автоматически сохраняются в CRM.

> CRM-формы доступны не на всех тарифах.

---

## Как создать CRM-форму

1. Перейдите в раздел **CRM > Продажи > Каналы продаж > CRM-формы**
2. Нажмите **Добавить**
3. Выберите сценарий — шаблон под конкретную задачу:
   - *Обратная связь* — сбор отзывов
   - *Онлайн-событие* — регистрация на вебинар
   - *Индивидуальная настройка* — свой вариант

## Как настроить CRM-форму

### Поля

Укажите, какие данные будет вводить пользователь:
- Имя, e-mail, телефон
- Удалите ненужные поля (крестик)
- Измените порядок (перетаскивание)
- Добавьте новые поля из элементов CRM

**Параметры поля:**

| Параметр | Описание |
|----------|----------|
| Название | Понятное название для похожих полей |
| Обязательное | Форма не отправится без заполнения |
| Множественное | Несколько значений (2 телефона) |
| Значение по умолчанию | Автоподстановка данных |
| Автозаполнение | Данные из браузера |
| Подсказка | Вспомогательный текст |

---

### Соглашения

Добавьте чекбоксы:
- Согласие на обработку персональных данных
- Ознакомление с условиями доставки

---

### Сущности в CRM

Выберите, какие элементы создаются при заполнении формы:
- Сделка + Клиент
- Счет
- Лид

**Правила обработки дубликатов:**

| Правило | Описание |
|---------|----------|
| Разрешить дубликаты | Всегда создается новая карточка |
| Заменять дубликаты | Данные обновляются из формы |
| Объединять дубликаты | Пустые поля заполняются, заполненные остаются |

> Дубликат — совпадают 2 из 3 параметров: ФИО, телефон, e-mail

---

### Заголовок и кнопка

- Создайте заголовок и подзаголовок
- Измените надпись на кнопке отправки
- Используйте теги персонализации
